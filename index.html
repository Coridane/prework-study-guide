HTML

  • The html element is the root of our HTML document.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The header element contains a navigation menu and/or introduction.
  • The footer element contains the author, copyright, sitemap, and navigation.

Git

  • git status shows what branch you're in and what files have been modified.
  • git add [filename] or -A will stage a file OR all files for your next commit.
  • git commit will commit staged content to your repository.
  • git clone [url] will retrieve files from the repo at that URL.
